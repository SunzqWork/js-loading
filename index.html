使用

在页面中引入必要的文件

<link rel="stylesheet" href="spin.css">
<script src="spin.js"></script>

基本使用,没有Text显示

HTML

<div id="demo" >
</div>

JavaScript

var spin =new SpinLoading({ text: false });

效果一

表格中使用（默认）

HTML

<div id="demo1" >
    <table>...</table>
</div>
<button type="button" onclick="myspin1.close()">关闭spin</button>
<button type="button" onclick="myspin1.show()">打开spin</button>

JavaScript

var myspin1 = new SpinLoading('demo1');
